# Historia de Usuario 3 - Exploración del menú

**Historia:**  
Como cliente quiero explorar el menú de cafés y productos de panadería para elegir lo que deseo comprar.

**Criterios de aceptación:**  
- El sistema debe mostrar una lista de productos con nombre, precio y descripción.  
- Debo poder filtrar por categorías (ej. café, panadería).  
- La información debe estar siempre actualizada.
