# Historia de Usuario 2 - Inicio de sesión

**Historia:**  
Como cliente quiero iniciar sesión con mi correo y contraseña para poder acceder a mi cuenta.

**Criterios de aceptación:**  
- El sistema debe validar credenciales correctas.  
- Si los datos son incorrectos debe mostrar un mensaje de error.  
- Mantener la sesión activa hasta que el usuario cierre manualmente.
