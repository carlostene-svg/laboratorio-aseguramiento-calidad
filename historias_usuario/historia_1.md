# Historia de Usuario 1 - Registro de usuario

**Historia:**  
Como cliente deseo registrarme con mi correo y contraseña para poder crear una cuenta en LePetitDep.

**Criterios de aceptación:**  
- El sistema debe validar que el correo sea único y correcto.  
- La contraseña debe tener mínimo 8 caracteres (Mayúscula,minúscula,caracter especial y número).  
- Al registrarme debo recibir un mensaje de confirmación al correo.
