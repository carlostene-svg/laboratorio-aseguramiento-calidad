# Historia de Usuario 5 - Realizar pedido

**Historia:**  
Como cliente quiero confirmar mi pedido y simular un pago para obtener un número de orden.

**Criterios de aceptación:**  
- El sistema debe mostrar un resumen del pedido antes de confirmar.  
- Debo poder elegir el método de pago simulado.  
- Al confirmar mi pedido se debe genera un número único de orden.
